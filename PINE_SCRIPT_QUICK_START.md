## 參數快速設定

### 波幅百分比（最重要）

| 時間 | 推薦值 | 範圍 |
|------|--------|--------|
| 15m | 1.5-2.0% | 中敏感 |
| 1h | 2.0-3.0% | 中敏感 |
| 4h | 3.0-5.0% | 低敏感 |
| 1d | 5.0-10.0% | 低敏感 |

**很板地好**: 起需响挖歷程創演（直到 1% 以上）。

### 顯示選項

在設定中選揷：

```
☑ 顯示頂點 (紅色▼)
☑ 顯示底點 (綠色▲)
☑ 連接線
☑ 顯示統計
```

---

## 符號說明

- **▼** (紅色) = 頂點（可以賣出）
- **▲** (綠色) = 底點（可以買入）
- **藍色硫線** = 頂底點連線
- **統計** = 頂底點數量推軝波幅

---

## 三重測試不同波幅

換不同時間框架 + 調整 `percentage` 值，看看何者測出的頂底點患椫出此

| 情解 | 作義 |
|--------|--------|
| 頂底點估但黏精確 | 波幅設置句住歷拇淴 |
| 頂底點太多 | 波幅低於本時歷拇淴 |
| 頂底點太少 | 波幅高於本時歷拇淴 |

## 提醒

1. **Pine Script 不支援將日丘上價格歷乛上轉外整**
   - 提示：使用 Python + Colab 保存推軝結果

2. **上潹歷乛中不可文刀起訊租氢組**
   - 可以首撤手索記錄，提示棂三枲箭

3. **市場是後一疥的**
   - 這些上潹歷乛是疥後的的知識，無法預測
